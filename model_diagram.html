03
Why lot_position matters
First wafer in a lot ≠ last wafer. Tool warm-up, pad wear, chamber conditioning all create within-lot drift in Vt. Position captures this without using the target.
Feature Assembly
Temporal Train / Val Split
Lots sorted by PARAM_END_DATETIME.
First 80% of lots → train
Last 20% of lots → validation

Entire lots stay together. Never split at wafer level — that leaks lot-level correlation.

Model
⚡ runs after last process step · before electrical test
CatBoost Classifier
Gradient boosted decision trees. Each tree asks yes/no questions about sensor values to separate outlier wafers from good ones. No sequence modeling — each wafer is a static snapshot of its entire process history.
